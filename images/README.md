# 📁 Dossier `images/`

Dépose tes visuels ici avec **exactement** les noms ci-dessous.
Le site les affiche automatiquement. Tant qu'un fichier est absent, un joli
placeholder (dimension + nom de fichier attendu) reste affiché à sa place.

> Format conseillé : **JPG** optimisé pour le web (qualité 80, < 300 KB par image).
> Pour les visuels avec transparence, tu peux utiliser du PNG (garde le même nom mais en `.jpg` → renomme l'extension dans le HTML si besoin).

> ⚠️ **Images de démo incluses** : les 40 `album-*-*.jpg` (4 albums × 10) sont des placeholders numérotés générés pour que la galerie soit cliquable tout de suite. **Remplace-les par tes vrais visuels en gardant exactement les mêmes noms de fichiers.** Tu peux aussi les supprimer si tu veux repartir de zéro (les albums redeviendront alors des placeholders non cliquables tant qu'ils sont vides).

**4 projets × 10 images** numérotées. Sur la carte fermée on voit les **3 premières images** (miniatures).
Au clic : grande image landscape + bande de toutes les images en bas (clic = affichage en grand, flèches ←/→).

| Projet   | Fichiers                                             |
|----------|------------------------------------------------------|
| Album_1  | `album-1-1.jpg` → `album-1-10.jpg` (les 3 premières = miniatures) |
| Album_2  | `album-2-1.jpg` → `album-2-10.jpg`                   |

> 🖼️ **Tous les formats sont acceptés dans un même album** : carré (1080×1080), paysage (1920×1080), A4/A5 portrait, etc. Le viewer s'adapte automatiquement au format de chaque image — au clic, l'image s'affiche **telle quelle, en entier, sans rognage**, et les miniatures reflètent leur format réel. Aucun redimensionnement à faire de ton côté.
